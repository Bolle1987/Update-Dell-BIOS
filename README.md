
# Update-Dell-BIOS.ps1
A powershell script that downloads and parses a couple xml files from downloads.dell.com to update the BIOS to the latest version.

> [!IMPORTANT]<br/>
> WARNING:</br>
> This script performs a BIOS update.</br>
> BIOS updates can render a device unusable.</br>
> Use at your own risk.</br>
> Ensure before use:</br>
> - You are running this on a Dell system</br>
> - The device is connected to reliable power</br>
> - Important data is backed up</br>
</br>

> [!NOTE]<br/>
> Direct PowerShell Command:</br></br>
> `iex (irm https://raw.githubusercontent.com/Bolle1987/Update-Dell-BIOS/master/Update-Dell-BIOS.ps1)`</br>
<br/>
